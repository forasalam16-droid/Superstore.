Advertising Performance Dashboard – Power BI


This Power BI project analyzes digital advertising campaign performance using data from campaigns, advertisements, users, and ad events. The dashboard provides insights into key metrics such as impressions, clicks, likes, shares, engagement, CTR, conversion rate, campaign budget, CPC, CPE, and ROI. It is designed with interactive visualizations and filters to help monitor campaign effectiveness, audience engagement, and overall advertising performance.
